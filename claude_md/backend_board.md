사용 흐름

1. 게시판 목록 → POST /portal/boards/list
2. 게시판 진입 → POST /portal/boards/posts/list { boardId, currentUser }
3. 글 상세 → GET /portal/boards/posts/{postId}?currentUser= + GET /portal/boards/files/post/{postId} + POST /portal/boards/comments/list
4. 좋아요 → POST /portal/boards/likes/toggle
5. 글쓰기 → GET /portal/boards/posts/next-id → POST /portal/boards/posts/save → 첨부 시 POST /portal/boards/files/upload
6.
